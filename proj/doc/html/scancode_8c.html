char getCharFromMakecode ( uint8_t  makecode)

Retrieves a character from a given makecode.

This function returns the corresponding character based on the provided makecode parameter and the current keyboard layout.

Parameters
makecodeThe makecode to retrieve the character from.
Returns
The corresponding character if found, ' ' (space) if makecode matches SPACE_SCANCODE, '
' (newline) if makecode matches ENTER_SCANCODE, or 0 if no matching character is found or the current keyboard layout is set to none.
