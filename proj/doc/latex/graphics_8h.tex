Sets up the frame buffer for the graphic mode. 

This function is called after the graphic mode has been set using \doxylink{graphics_8h_a967432a6c8128a907b937b0f3f4b4472}{set\+\_\+graphic\+\_\+mode()}. It sets up the frame buffer by allocating physical and virtual memory for the frame buffer based on the mode information obtained from \doxylink{graphics_8h_a967432a6c8128a907b937b0f3f4b4472}{set\+\_\+graphic\+\_\+mode()}. It also allocates memory for the frame buffer using {\ttfamily malloc()}.

\begin{DoxyNote}{Note}
This function should only be called after calling {\ttfamily \doxylink{graphics_8h_a967432a6c8128a907b937b0f3f4b4472}{set\+\_\+graphic\+\_\+mode()}}.
\end{DoxyNote}
\begin{DoxyReturn}{Returns}
0 if the frame buffer was set up successfully, otherwise 1 if an error occurred. 
\end{DoxyReturn}
