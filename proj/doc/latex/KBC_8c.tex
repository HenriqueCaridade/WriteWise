Writes a command byte to the Keyboard Controller input buffer. 

This function writes a command byte to the input buffer of the keyboard controller using the specified port. It checks the status byte of the keyboard controller to ensure that the input buffer is not full. If the input buffer is not full, the command byte is sent to the specified port with sys\+\_\+outb.
